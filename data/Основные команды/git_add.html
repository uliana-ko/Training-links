$ git add

Команда добавляет файл(ы) в область staged
Например: $ git add file.txt => добавить только file.txt
$ git add folder => добавить папку folder и все содержимое папки
$ git add –A => с флагом –A добавит все файлы в область staged
$ git add . => более сокращенный аналог команды выше, добавит все файлы по текущему пути
