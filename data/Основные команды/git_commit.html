$ git commit

У каждого коммита есть свое описание(комментарий). Комментарий описывает, какие изменения в проекте вошли в коммит (что было добавлено, реализовано, удалено и т.д.)

Для добавления комментария используется флаг –m
$ git commit –m “Fixed bug with subscription form”
Комментарий должен быть абсолютно у каждого коммита.
